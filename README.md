# 🔥linktree-clone

![linktree-clone.gif](resources/readme/linktree-clone.gif)

## 📖Sobre

"Clone" de algumas funcionalidades da plataforma Linktree, desenvolvido para praticar conceitos de frontend e backend.

## 🛠Tecnologias utilizadas

- Laravel
- VueJS
- TailwindCSS
- InertiaJS

## 🎮Rodando o projeto

```bash
php artisan migrate
npm run dev
php artisan serve
```
